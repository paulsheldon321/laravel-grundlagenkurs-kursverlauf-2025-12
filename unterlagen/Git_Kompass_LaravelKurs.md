# Git-Kompass – Laravel Grundlagenkurs

## Basisbefehle

```bash
git status
git add .
git commit -m "Nachricht"
git log --oneline
```

## Remote (falls genutzt)

```bash
git remote -v
git push
git pull
```

## Typische Schritte im Kurs

### Änderungen sichern

```bash
git add .
git commit -m "Tag 1 abgeschlossen"
```

### Stand anzeigen

```bash
git status
```

### Differenzen ansehen

```bash
git diff
```

## Tags (Trainer)

```bash
git tag lg-t1-end
git tag lg-t2-end
git tag lg-t3-end
git tag lg-t4-end
git tag lg-t5-end
```

## Tipps

- Häufig kleine Commits machen.
- Commit-Nachrichten kurz, aber verständlich halten.
- Nicht „alles kaputt“ pushen – erst lokal testen.
